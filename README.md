# GSM
Проект по полному переносу аудиоконтента из World of Tanks и Мир Танков в Tanks Blitz.

-----------------------------  

### В чем уникальность модификации?
- Отличное качество звука.
- Полностью оригинальные настройки громкости и иеархия Wwise прямо из World of Tanks.
- Продвинутая музыкальная система.
- Боевая музыка состоит из отдельных партий, которые случайно миксуются между собой. Каждая сыгранная боевая тема уникальна.
- Почти полная замена звуков.
- Реверберация и ранние отражения.
- Точное позиционирование.
- По сравнению с оригинальной игрой, увеличенная информативность звуков.
- Случайность, неповторимость звуков.

### Контент модификации:
- Интерфейс
  - Кнопки интерфейса
  - Ангарный интерфейс
  - Боевой интерфейс
    - Критические повреждения модулей
- Техника
  - Столкновения танков
  - Движение танков
    - Двигатели
      - Эффект поврежденного двигателя
    - Гусеницы
    - Трансмиссия
    - Тряска танка
    - Поверхность под гусеницами
    - Ветер, задувающий камеру
  - Ракетные ускорители
- Голосовые уведомления
  - Озвучка экипажа
  - Озвучка боевой коммуникации
- Эффекты
  - Уничтожение танков
  - Выстрелы
  - Попадания по танкам
  - Попадания по поверхности
  - Заряжание
  - Пролет снарядов
  - Объекты разрушения
  - Боевая атмосфера
- Окружающая среда
  - Эмбиент в ангаре
  - Эмбиент на вкладках в ангаре
  - Эмбиент в бою
- Музыка в бою
  - Предбоевая музыка
  - Генеративная боевая музыка
  - Послебоевая музыка (зависит от карты, на которой проведен бой)
- Музыка в ангаре
  - Генеративная музыка в ангаре
  - Генеративная музыка на вкладках в ангаре

### Инструменты, использующиеся при разработке:
- [wwiser](https://github.com/bnnm/wwiser) - парсинг банков `.bnk` оригинальной игры, восстановление иеархий из банка в проект Wwise.
- [FusionTools](https://www.nexusmods.com/devilmaycry5/mods/1870) - извлечение аудиофайлов в формате Wwise Encoded Media `.wem` из банков `.bnk` и `.pck` World of Tanks.
- [vgmstream](https://github.com/vgmstream/vgmstream) - конвертация аудиофайлов формата `.wem` в `.wav`.
